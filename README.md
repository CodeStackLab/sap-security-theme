# AdSense Approved Health Blog Theme

**Live Demo**: [https://healthbeyondage.com/](https://healthbeyondage.com/)

A modern, fast, and SEO-friendly custom WordPress theme built specifically for the **health, wellness, and medical blog niches**. This theme is optimized to help you get **Google AdSense approval** quickly by providing a clean, professional, and lightning-fast user experience without the bloat of page builders.

## Features
- **100% Custom Built**: No Elementor, Gutenberg blocks, or heavy builders.
- **Lightning Fast**: Optimized code with minimal external dependencies.
- **Customizer Integrated**: Manage logo, copyright, about section, and social media links directly from the WordPress Customizer (`Appearance > Customize`).
- **Responsive Design**: Flawless experience on desktop, tablet, and mobile devices.
- **Dynamic Content**: Uses native WordPress posts, categories, and custom templates for seamless content management.

## Installation Guide

1. **Download the Theme**:
   Download this repository as a `.zip` file.
2. **Upload to WordPress**:
   Log into your WordPress Admin Dashboard.
   Navigate to `Appearance > Themes > Add New > Upload Theme`.
   Upload the `healthbeyondage-theme.zip` file and click "Install Now".
3. **Activate**:
   Once installed, click "Activate".

## Setting up the Theme

### 1. Front Page Setup
1. Create a new page named "Home" (or whatever you prefer) and set its template to **Default Template**.
2. Create another new page named "All Articles" (or "Blog").
3. Go to `Settings > Reading`.
4. Under "Your homepage displays", select **A static page**.
5. Set **Homepage** to "Home" and **Posts page** to "All Articles".

### 2. About Page Setup
1. Create a new page named "About Us".
2. In the right sidebar, under **Page Attributes**, set the **Template** to **About Us Page**.
3. Publish the page.

### 3. Customizing the Theme (Appearance > Customize)
Navigate to `Appearance > Customize` to configure theme settings:

- **Site Identity**: Upload your site logo here.
- **Theme Settings**: 
  - Edit the Footer "About" text.
  - Set your social media URLs (Facebook, X/Twitter, YouTube, Pinterest).
  - Update the Copyright text.
- **Trust Metrics**: Update the stats shown in the trust bar (e.g., "150+ Expert Articles", "100% Medically Reviewed").

## Development
This theme uses standard PHP, HTML, Vanilla CSS, and JavaScript. 
- All CSS is located in `assets/css/main.css`.
- Core functionalities and hooks are registered in `functions.php`.
- Customizer settings are handled in `inc/customizer.php`.
